Research Assistant RAG

Neo4j-backed PDF ingestion and contextual retrieval.

How To Use

  1. Click Sync PDFs to mirror Google Drive Paperpile files into local pdfs/.
  2. Choose ingest mode: test3 for first 3 PDFs, all for full corpus, or custom with explicit file paths.
  3. By default, already ingested PDFs are skipped. Enable Override Existing to reprocess them.
  4. Run Ingest to add/update graph data. Existing Neo4j data is never erased. Unreadable files are skipped and shown in failed_pdfs.
  5. Use Stop on Sync/Ingest/Query to cancel a running action.
  6. Ask a retrieval query in Search to get chunks with author and citation context.
