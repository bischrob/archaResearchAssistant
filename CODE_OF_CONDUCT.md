# Code of Conduct

## Our Standard

Participation in this project should be respectful, direct, and constructive.

Examples of expected behavior:

- giving actionable technical feedback
- disagreeing without hostility
- focusing on the work rather than the person
- respecting different levels of experience

Examples of unacceptable behavior:

- harassment or personal attacks
- abusive language
- deliberate disruption
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code of conduct.

## Reporting

If you experience or observe unacceptable behavior, report it privately to the maintainer rather than escalating publicly in a hostile thread.
